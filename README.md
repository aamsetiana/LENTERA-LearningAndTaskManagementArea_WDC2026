# Lentera - Learning And Task Manajement Area

**Lentera** adalah platform produktivitas digital yang dirancang untuk membantu pengguna mengelola fokus, tugas, dan catatan dalam satu ekosistem yang intuitif. Proyek ini dikembangkan sebagai _Final Submission_ untuk ajang **Web Development Competition (WDC) 2026**.

## 🚀 Fitur Unggulan

### 1. 📈 Pusat Kendali (Dashboard)

Visualisasi statistik produktivitas secara interaktif dan _real-time_.

- **Status Tugas**: Memantau antrean tugas aktif secara presisi.
- **Persentase Target**: Indikator progres penyelesaian tugas harian.
- **Beban Mingguan**: Grafik distribusi deadline dari Senin hingga Minggu.

### 2. ⏳ Meja Kerja (Workstation)

Ruang kerja minimalis untuk mencapai kondisi _deep work_.

- **Pomodoro Timer**: Pengatur waktu fokus yang terintegrasi dengan penghitung sesi otomatis.
- **Focus Ambient**: Fitur pemutar musik latar untuk meningkatkan konsentrasi.
- **Kanban Board**: Pengelolaan tugas dengan sistem _Drag-and-Drop_ (Rencana, Dikerjakan, Selesai).

### 3. 📊 Analisis Statistik (Analytics)

Evaluasi mendalam mengenai kebiasaan produktivitas pengguna.

- **Durasi Fokus**: Grafik batang yang merekam jam fokus harian.
- **Distribusi Tugas**: Grafik lingkaran untuk melihat porsi status seluruh tugas.
- **Tren Aktivitas**: Grafik garis yang memantau konsistensi input tugas setiap harinya.
- **Performa Mingguan**: Metrik khusus untuk memantau Hari On-Track, Rata-rata Tugas, dan Efisiensi Belajar.

## 🛠️ Tech Stack

- **Framework**: React.js (Vite)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Data Visualization**: Chart.js & React-Chartjs-2
- **Database**: Browser Local Storage (Data tersimpan secara lokal di perangkat pengguna)

---

## 📱 Optimalisasi Perangkat (Responsivitas)

Lentera telah diuji dan dioptimalkan untuk berbagai ukuran layar:

- **Desktop**: Tampilan maksimal untuk produktivitas penuh.
- **Tablet (iPad)**: Layout grid adaptif (2 kolom) yang mencegah elemen tumpang tindih.
- **Mobile**: Antarmuka ringkas dengan sistem navigasi yang efisien.
