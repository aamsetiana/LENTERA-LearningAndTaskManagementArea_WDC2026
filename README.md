# LENTERA (Learning & Task Management Area)
**Tim:** SEMOGA JUARA
**Universitas:** Universitas Kuningan

## Deskripsi
LENTERA adalah solusi berbasis web yang dirancang untuk memberdayakan mahasiswa dalam mengelola produktivitas akademik. Fokus pada antarmuka yang bersih dan fitur esensial untuk membantu mahasiswa tetap fokus dan teratur.

## Fitur Utama
1. **Dasbor Fokus**: Pengatur waktu belajar dengan latar suara atmosfer.
2. **Papan Tugas**: Manajemen tugas visual sistem Kanban.
3. **Catatan Kuliah**: Pencatatan ringkasan materi kuliah.
4. **Grafik Progres**: Visualisasi statistik tugas yang diselesaikan.
5. **Peta Jalan Harian**: Daftar prioritas utama hari ini.

## Teknologi
- Frontend: [React.js / HTML & CSS / Tailwind]
- Deployment: Vercel
